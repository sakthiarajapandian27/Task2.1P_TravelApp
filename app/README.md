# Travel Companion App- Task 2.1P

## Overview
This is an Android application developed as part of the SIT708 unit Task 2.1p. The app supports conversions for Currency, Fuel Efficiency, and Liquid Volume

## Features
- Currency conversion (USD, AUD, EUR, JPY, GBP)
- Fuel conversion (mpg to km/L and vice versa)
- Temperature conversion (Celsius, Fahrenheit, Kelvin)

## Technologies Used
- Java
- Android Studio
- XML for UI design

## How to Use
1. Select a category (Currency, Fuel, or Temperature)
2. Choose the source unit
3. Choose the destination unit
4. Enter the value
5. Click the Convert button to see the result

## Validation
- Prevent the app from processing empty or non-numeric inputs (shows a Toast or Error message).
- Handle “Identity Conversions” (e.g., converting USD to USD) by notifying the user or returning the same
value.
- The app handles negative values appropriately (e.g., fuel efficiency cannot be negative)
